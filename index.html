01

ホーム画面からタップ1つでタスク管理

確認・追加・完了 — 全部この画面だけ

You
あなた
ホーム画面に追加したアイコンをタップ
ログイン不要。アプリのように即起動
⚙
司令官（GAS）
TASKS.mdを読み込み、最新データで画面を構築
昨日の完了タスクは0時に自動で履歴へ移動済み
📺
Web App
今日のタスク・進捗・アイデア・知識リンク…全部がここに
追加・完了チェック・並び替え、全てこの画面で完結。操作するたびにスプシもTASKS.mdも裏で自動同期
進捗: 3 / 21完了 — チェックするたびにバーが伸びる
Point — タップ1回で起動。確認も追加も完了も全部この画面だけ。操作するたびに裏で全データが自動同期されている。
02

Cowork（AI）で編集 → アプリに自動反映

PCで指示するだけ。スマホを開いたらもう反映されてる

You
あなた（PCでCoworkに指示）
「TASKS.mdにこのタスク追加して」と一言伝える
Cowork（AI）に話しかけるだけ。ファイルを開く必要すらない
AI
Cowork（AI）
TASKS.mdを開いて、適切な場所にタスクを追記
プロジェクトの構造を理解して、正しいセクションに自動で追加
### Todoアプリ
- [ ] GAS＋Gemini試作
- [ ] 新しいタスクをここに追加
⚙
司令官（GAS）— 次回アプリ起動時
TASKS.mdの変更を検知 → スプシに自動反映
Coworkが書き込んだタスクが、masterシートに自動で取り込まれる
📱
あなた（スマホでアプリを開く）
さっきCoworkに頼んだタスクが、もうアプリに表示されてる
自分では何も操作してないのに、タスク一覧に新しいタスクが増えている
ここがすごい — AIに話しかけるだけでタスク管理が動く。TASKS.md → スプシ → アプリが全部つながっているから、どこから編集しても全体に反映される。逆にアプリで完了チェックすればTASKS.mdにも[x]がつく。双方向同期だからこそできる技。
03

寝てる間にシステムが全部片付けてくれる

0時に自動で「翌日の準備」が完了している

⏰
トリガー（毎日0時に自動発動）
日次リセット処理を開始
あなたが寝ている間に、GASが自動で動き出す
⚙
司令官（GAS）
4つの処理を全自動で実行
1. 完了タスク → 履歴シートに保存してtodayから削除
2. 未完了タスク → そのまま翌日に引き継ぎ
3. 繰り返しタスク →「未着手」に戻して翌日もリストに残す
4. TASKS.mdにも最新状態を書き戻し
📋
スプレッドシート（記録庫）
履歴にいつ何を完了したか全て記録
1ヶ月後に「先月何やったっけ？」→ 履歴を見れば全部わかる
☀
翌朝のあなた
アプリを開くと、もう今日やるべきタスクだけが並んでいる
昨日の残り + 毎日の繰り返しタスクが自動セット。朝の「整理の時間」がゼロ
ここがすごい — 「毎日の片付け」「翌日の準備」「繰り返しタスクの再セット」を全部自動でやってくれる。人間がやることは朝アプリを開くだけ。タスク管理の雑務がゼロになる。しかも履歴が残るから、振り返りもできる。
